Online Demo

Upload a single image to try ChemSchematicResolver online. Alternatively, check out one of our example files.

Please note that this demo cannot handle HTML / XML inputs, or directories containing multiple images. Install ChemSchematicResolver to access the full version of this tool with these capabilities. If extracting from a document, please use HTML or XML from the supported publishers: the Royal Society of Chemistry, Elsevier and Springer. To extract from articles by other publishers, please download the ChemDataExtractor toolkit and write your own document reader.

Supported image formats: BMP, DIB, GIF, JPEG, PNG, PPM, SGI, TGA, TIFF. Large images may not be resolved in our demo: please install ChemSchematicResolver locally for these cases.

This demo performs best on high resolution images with distinct gaps between objects in the image. Images must only contain only chemical diagrams and text labels.
